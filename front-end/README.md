# sistema-leilao

## Preparação do Front-end

```
npm install
```

### Para rodar o servidor use o comando.
```
npm run serve
```
### Tecnologias utilizadas
    axios: faz requisições para a porta 8080
    bulma: framework de css
    vue-router: mapeamento das rotas
